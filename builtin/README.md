# Built-in to Xteink (Stock Firmware)
**Note** this replaces the original dashboard.

1. **Download** both `GCSBS.html` and `jszip.min.js`.
2. **Head** to `http://192.168.3.3/`.
3. **Toggle** SD card option to **disabled**.
4. **Upload** the `GCSBS.html` and `jszip.min.js` files into `system` folder directory.
5. **Reload** from `http://192.168.3.3/`.
6. **Re-toggle** SD card option to **enable**.
